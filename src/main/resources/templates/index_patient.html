Cadastrando Pacientes


Para cadastrar um novo paciente siga o procedimento:

→Clique no botão add

*Será aberto a página de Cadastro

→Preencha o campo Nome com o nome do paciente

*Este nome ficará disponivel para que seja buscado posteriormente no sistema

→Clique no campo Sobrenome e preencha com o sobrenome do paciente

→Clique no campo Sexo e selecione o sexo do paciente

→Clique no campo Data de Nascimento e preencha com a data de nascimento do paciente

→Clique no campo Religião e preencha com a religião do paciente

→Clique no campo Estado civil e selecione o estado civil do paciente

→Clique no campo Cidade e preencha com a cidade do paciente

→Clique no campo Unidade Federativa e preencha com a Unidade Federativa do paciente

→Clique no campo CEP e preencha com o CEP do paciente

→Clique no campo Endereço e preencha com o Endereço do paciente

→Clique no campo Telefone fixo e preencha com o Telefone fixo do paciente

→Clique no campo Telefone celular e preencha com o Telefone celular do paciente

→Clique no campo Email e preencha com o Email do paciente

→Clique no campo CPF e preencha com a CPF do paciente

*deve ser utilizado um CPF válido

→Clique no campo RG e preencha com o RG do paciente

→Clique no campo Antecedentes e selecione doenças que o paciente já teve

*caso não tiver, pode deixar em branco

→Clique no campo Medicamentos e selecione medicamentos que o paciente utiliza

*caso não tiver, pode deixar em branco

→Clique no botão Salvar para cadastrar o paciente

*ou

→Clique no botão Voltar para não cadastrar o paciente, a página retornará para a lista de pacientes

Fechar

Editando Pacientes


Para editar um paciente siga o procedimento:

→Na linha do paciente que quer editar, Clique no botão edit

*Será aberto a página do paciente cadastrado

→Clique no(s) campo(s) do paciente que deseja editar

*Faça as alterações necessárias

→Clique no botão Salvar para editar o paciente

*ou

→Clique no botão Voltar para não editar o paciente, a página retornará para a lista de pacientes

Fechar

Consultando Pacientes


Para Consultar um paciente siga o procedimento:

→Clique no campo Nome e preencha o nome, ou parte, para consultar o nome do paciente

→Clique no campo CPF e preencha o CPF do paciente para consultar

→Clique no botão search para pesquisar

Fechar

Removendo Pacientes


Para Remover um paciente siga o procedimento:

→Na linha do paciente que quer remover, Clique no botão delete

*Será aberto uma janela

→Clique no botão OK para remover o paciente

*ou

→Clique no botão Cancelar para não remover o paciente, a página retornará para a lista de pacientes

Fechar

Imprimindo Ficha Paciente


Para Imprimir uma Ficha do paciente siga o procedimento:

→Na linha do paciente que quer imprimir a Ficha, Clique no botão local_printshop

*Será aberto uma janela de impressão do seu sistema

Fechar

Limpar Campo de Pesquisa


Para Limpar o campo de pesquisa de um paciente siga o procedimento:

→Na linha de Pesquisa, Clique no botão clear_all

*Será limpo o campo nome e cpf
