Cadastrando Mendicamentos


Para cadastrar uma novo Medicamento siga o procedimento:

→Clique no botão add

*Será aberto a página de Cadastro

→Clique no campo nome e preencha com o nome e dados do medicamento

→Clique no botão Salvar para cadastrar o Medicamento

*ou

→Clique no botão Voltar para não cadastrar o Medicamento, a página retornará para a lista de medicamentos

Fechar

Editando Medicamentos


Para editar um Medicamento siga o procedimento:

→Na linha de medicamentos que quer editar, Clique no botão edit

*Será aberto a página do medicamento cadastrado

→Clique no campo do nome se deseja editar e realize as alterações

→Clique no botão Salvar para editar o medicamento

*ou

→Clique no botão Voltar para não editar o Medicamento, a página retornará para a lista de medicamentos
