Cadastrando Usuários


Para cadastrar um novo usuário siga o procedimento:

→Clique no botão add

*Será aberto a página de Cadastro

→Preencha o campo Nome com o login do Usuário

*Este login não deve conter numeros e deve ser diferente de qualquer outro cadastrado no sistema

→Clique no campo Senha e preencha com uma senha

→Clique no botão Salvar para cadastrar o usuário

*ou

→Clique no botão Voltar para não cadastrar o usuário, a página retornará para a lista de usuários

Fechar

Editando usuário


Para editar um usuário siga o procedimento:

→Na linha do usuário que quer editar, Clique no botão edit

*Será aberto a página do usuário cadastrado

→Clique no(s) campo(s) do usuário que deseja editar

*Faça as alterações necessárias

→Clique no botão Salvar para editar o usuário

*ou

→Clique no botão Voltar para não editar o usuário, a página retornará para a lista de usuário
